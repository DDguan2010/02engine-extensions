SeaCloud Extension 文档

版本 1.0 | 发布日期: 2025年8月13日

用法

  1. 使用以下块：
    • 连接到服务器 - 建立 wss://seacloud-s.deep-sea.filegear-sg.me 连接。
    • 注册用户 - 注册 (用户名, 密码)。
    • 登录用户 - 登录 (用户名, 密码)。
    • 验证开发者 - 验证 (当前账户: dev_user / dev_pass)。注意：此密码可能在未来版本中修改。
    • 上传文件 - 上传 (用户名, ID, 内容)。
    • 获取文件 - 获取 (用户名, ID)。
    • 删除文件 - 删除 (用户名, ID)。
    • 替换文件 - 替换 (用户名, ID, 新内容)。
    • 检查用户存在 - 检查 (用户名)。
    • 检查登录状态 - 检查 (用户名)。
    • 是否连接 - 检查连接状态。
  2. 确保服务器在线(有很小概率被墙)
  3. 本项目使用AGPL 3.0协议
  4. 有些获取可能要等待0.几秒才有效果
